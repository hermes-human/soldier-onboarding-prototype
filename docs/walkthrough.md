# Soldier Worker Onboarding Prototype Walk-through

## Unlock
- Password: `Soldier01`

## Admin demo path
1. Enter the password gate.
2. Choose **I'm Admin**.
3. Start on the dashboard — this is the main overview / wow screen.
4. Open **Forms** and click **Samuel Nguyen** to show a submitted onboarding pack.
5. Use **Approve**, **Request Changes**, or **Reject** to show in-session status changes.
6. Open **Workers** and click any worker to show profile, licences, activity, and notes.
7. Open **Invite** and create a sample worker to preview the outbound SMS copy.

## Worker demo path
1. Return to the role select screen.
2. Choose **I'm a Worker**.
3. Start from **Sam Example**.
4. Open the dashboard and show the progress / step status cards.
5. Enter the onboarding wizard.
6. Step through the 10 sections.
7. On upload fields, click **Add file** to simulate uploads.
8. On signature fields, click **Simulate signature**.
9. Submit the form.
10. Switch back to admin mode and open the completed forms list to review the submission.

## What to point out during the demo
- worker side is mobile-first but still desktop-friendly
- admin side is desktop-first but remains mobile-usable
- all data is simulated in-session only
- the yellow prototype ribbon is intentional so nobody mistakes this for production
